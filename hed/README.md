# conf

学習パターンを制御するための、音素や楽譜情報の分類ファイル。

どれか1つのファイルを使って学習させます。

## hed を作るときの注意

- d1, e1, f1 はCQSの最初に置かないといけない。

## ファイル比較

いずれも kiritan-singing および sinsy のjp_qst001.hed （2020年10月以前のもの）を改造したもの。
